# The-Intruder-Mineshaft
My OP Script I made for The Intruder's Mineshaft map with the power of Chat GPT! (Read the readme for more information)

Hello, I am BMF.

I made this OP script GUI for ROBLOX The Intruder's multiplayer mineshaft map with a mixture of playing the game to see how it works, using Dex Explorer, and the power of Chat GPT.

Please note that while this script may seem like an unfair advantage, most of it is actually just client sided to help you win. 

Features:

- Infinite Yield
- Dark Dex
- Win Game (WIP)
- God Mode (WIP)
- Infinite Stamina (WIP)
- Full Bright
- Intruder ESP
- Generator ESP
- Return to Lift
